# 项目概述

## 1. 项目名称

基于深度学习的单字图像识别系统  
（使用 MNIST 的初始化版本）

---

## 2. 项目背景

单字图像识别是计算机视觉中的典型图像分类任务，其核心目标是：输入一张仅包含一个字符的图像，输出其对应的类别标签。

该任务在以下场景中具有广泛应用价值：

- 手写数字识别
- 印刷字符识别
- 古文字识别
- 文档数字化
- 文化遗产保护

本项目当前以 **MNIST** 作为初始数据集，构建一个完整、可复用的识别流程。项目重点覆盖以下完整流程：

- 数据加载
- 模型构建
- 模型训练
- 模型评估
- 模型保存
- 单张图像推理
- 结果可视化

之所以选择 MNIST，是因为其结构简单、标准化程度高，适合快速搭建项目原型。在后续阶段，可以在不改变整体系统架构的前提下，将数据集替换为自定义的小篆单字数据集。

---

## 3. 项目目标

本项目的主要目标如下：

1. 构建一个结构清晰、可扩展的单字识别框架。
2. 使用 MNIST 完成基础的单字符分类任务。
3. 使用 PyTorch 实现训练、验证、测试与推理流程。
4. 生成基础可视化结果，如损失曲线和准确率曲线。
5. 为后续迁移到小篆单字识别预留统一接口。

---

## 4. 项目范围

本项目当前仅聚焦于以下内容：

- **单字识别**
- **图像分类任务**
- **MNIST 数据集**
- **PyTorch 实现**
- **命令行推理**

本项目当前**不包括**以下内容：

- 文本检测
- OCR 行级识别
- 多字符序列建模
- 第一阶段的 GUI 部署

---

## 5. 核心思路

本项目的核心思路是：

> 先在 MNIST 上构建一个完整、稳定、可复用的单字识别流程，  
> 后续再将数据集替换为自定义的小篆单字图像。

这种设计可以降低开发难度，提高工程实现效率。

---

## 6. 预期成果

完成当前初始化版本后，项目应提供以下成果：

1. 一个可运行的 MNIST 分类模型
2. 一套完整的 PyTorch 训练流程
3. 一个保存好的最优模型权重文件
4. 一个单图推理脚本
5. 基础可视化结果
6. 一个可复用的项目结构，便于后续替换数据集

---

## 7. 技术栈

- **编程语言：** Python
- **深度学习框架：** PyTorch
- **数据集工具：** torchvision
- **图像处理：** PIL / torchvision.transforms
- **可视化：** Matplotlib
- **评估支持：** scikit-learn
- **进度显示：** tqdm

---

## 8. 当前阶段开发目标

当前阶段的目标是实现一个最小可运行版本（MVP），具体包括：

- 成功加载 MNIST
- 训练一个 CNN 模型
- 评估分类准确率
- 保存最优模型
- 实现单张图像预测
- 生成训练曲线

当该 MVP 完成后，项目即可平滑迁移到小篆单字识别任务。
