# 项目概览

SealScriptOCR 当前定位为 **秦简牍动物字体识别** 的轻量级视觉分类基线。项目将单个动物相关字形或符号图像识别建模为多类别图像分类任务：输入一张已裁剪的单字符图像，输出对应的类别标签。

本文档说明项目目标、能力边界与阶段路线。训练和推理命令见 [05_Training_and_Inference_Guide.md](05_Training_and_Inference_Guide.md)。

## 范围

当前阶段只做单图分类：

- 输入：一张单字符或单符号图像。
- 输出：一个类别标签及各类别概率。
- 数据：秦简牍动物字体图像，按类别文件夹组织。
- 模型：`SimpleCNN` 与单通道 `ResNet18`。
- 运行方式：命令行训练与单图推理。

暂不包含：

- 文本检测、字符切分或整行 OCR。
- 多字符序列识别。
- Web、API 或 GUI 服务。
- 大规模预训练或在线标注系统。

## 产品目标

项目目标不是追求复杂系统，而是建立一条可信、可复现、易扩展的研究基线：

1. 快速验证秦简牍动物字体图像能否被 CNN 稳定分类。
2. 建立统一的数据目录、预处理、训练、评估和推理流程。
3. 产出可比较的模型权重、训练曲线和实验记录。
4. 为后续更强模型、数据增强、混淆分析和批量推理留出清晰接口。

## 当前能力

| 模块 | 状态 |
| --- | --- |
| 自定义数据集 | 支持 ImageFolder 结构：`train/val/test/<class>/*` |
| 图像预处理 | 灰度化、自动对比度、中值滤波、等比例缩放并填充 |
| 模型训练 | 交叉熵损失、Adam、按验证集准确率保存最优权重 |
| 模型结构 | `SimpleCNN`、单通道 `ResNet18` |
| 推理 | `python src/infer.py --image ...` 单图推理 |
| 输出产物 | `best_model.pth`、`training_plot.png`、`label_map.json` |

## 目标数据集

推荐将每个动物相关字形作为一个类别，例如：

```text
data/
├── train/
│   ├── 马/
│   ├── 牛/
│   └── 犬/
├── val/
│   ├── 马/
│   ├── 牛/
│   └── 犬/
└── test/
    ├── 马/
    ├── 牛/
    └── 犬/
```

类别名可以使用汉字、释读名、编号或拼音，但同一数据集内必须稳定一致。

## 设计原则

- **真实基线**：文档只描述当前代码真实支持的能力。
- **先小后大**：先保证可训练、可复现、可推理，再扩展评估和部署能力。
- **数据优先**：优先修正数据质量、类别定义和样本平衡，再堆叠模型复杂度。
- **默认可复现**：随机种子、配置、权重和输出路径应可追踪。
- **易于替换**：数据集、模型和预处理可逐步替换，不破坏训练主流程。

## 路线图

- 增加训练配置的命令行参数。
- 增加测试集评估与混淆矩阵导出。
- 为简牍图像差异增加受控数据增强。
- 增加批量推理与 CSV 输出。
- 增加实验元数据记录。
- 在数据质量稳定后评估更强模型骨干。
