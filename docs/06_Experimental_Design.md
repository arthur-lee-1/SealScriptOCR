# 实验设计

本文档定义秦简牍动物字体分类的第一批实验。

## 目标

在加入复杂模型前，先建立可靠基线：

1. 确认数据集可以训练分类器。
2. 对比 `SimpleCNN` 与 `ResNet18`。
3. 找出容易混淆的类别和薄弱数据。
4. 建立可复用的实验记录格式。

## 任务定义

| 项目 | 定义 |
| --- | --- |
| 输入 | 单张裁剪好的字形图像 |
| 输出 | 动物字形类别 |
| 任务类型 | 多类别图像分类 |
| 主要指标 | 验证集准确率 |
| 次要指标 | 每类准确率、混淆矩阵 |

次要指标属于计划项；当前代码主要输出损失和准确率。

## 基线实验

### E1：流程冒烟测试

目的：验证项目可以端到端运行。

| 设置 | 值 |
| --- | --- |
| 模型 | `simple_cnn` |
| 输入尺寸 | `28` |
| 训练轮数 | `3-5` |
| 预期结果 | 程序不崩溃，权重成功保存 |

### E2：SimpleCNN 基线

目的：建立快速基线。

| 设置 | 值 |
| --- | --- |
| 模型 | `simple_cnn` |
| 输入尺寸 | `28` |
| 优化器 | Adam |
| 学习率 | `1e-3` |
| 输出 | 最佳验证准确率和训练曲线 |

### E3：ResNet18 基线

目的：测试更强特征提取能力是否有帮助。

| 设置 | 值 |
| --- | --- |
| 模型 | `resnet18` |
| 输入尺寸 | `64` 或 `128` |
| 优化器 | Adam |
| 学习率 | `1e-3` |
| 输出 | 最佳验证准确率和训练曲线 |

### E4：数据质量对照

目的：衡量数据清洗是否比模型替换带来更大收益。

对比清洗前后：

- 删除重复样本
- 修复错误标签
- 移除不可辨认裁剪图
- 平衡类别数量
- 统一裁剪边距

## 数据划分

使用稳定的数据划分：

| 划分 | 用途 |
| --- | --- |
| train | 模型学习 |
| val | 选择最优权重和比较模型 |
| test | 最终报告，只使用一次 |

不要在测试集上调超参数。

## 指标

当前已有：

- 训练损失
- 验证损失
- 训练准确率
- 验证准确率

计划增加：

- 测试集准确率
- 每类准确率
- 混淆矩阵
- top-k 准确率
- 错误样本可视化

## 失败样本分析

对每个失败类别检查：

- 标签是否正确？
- 裁剪是否居中？
- 笔画是否缺失？
- 字形是否与另一类更接近？
- 该类别样本是否偏少？
- 训练集和验证集风格是否不同？

秦简牍图像常见问题：

- 墨迹褪色
- 笔画断裂
- 背景纹理明显
- 裁剪过窄或拉伸
- 动物字形变体视觉相似

## 决策规则

- 如果训练准确率很低，先检查预处理和标签。
- 如果训练准确率高但验证准确率低，优先处理过拟合和数据划分。
- 如果只有少数类别失败，优先做类别级数据清洗。
- 如果 `ResNet18` 整体提升明显，将其作为下一阶段基线。
- 如果 `ResNet18` 很快过拟合，回到数据清洗和数据增强。
